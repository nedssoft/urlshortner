### URL SHORTNER

- A simple App for shortening and expanding URLs

### Usage
## Shorten URL
- Enter a long URL to shorten
- click on shortend
- Wait for the result
- Copy and use

## Expand URL
- Enter a shortened URL to expand
- click on expand
- Wait for the result
- Copy and use

### Tools used
- [Google API](https://www.googleapis.com/urlshortener/v1/url)
